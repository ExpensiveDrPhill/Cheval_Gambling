## How to Run the Game
1. Clone repository ini
   ```bash
   git clone https://github.com/ExpensiveDrPhill/Cheval_Gambling.git

2. Install dependency yang diperlukan:
   ```bash
   pip install -r requirements.txt

3. **Jalankan Game**
    ```bash
    python -m core.GameManager
